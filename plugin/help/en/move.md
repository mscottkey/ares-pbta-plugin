> **+move** — Trigger a named move

**Usage**
```
+move <move name>
```

Triggers the named move, rolls the appropriate stat, and emits the move description. You must have the move in your Playbook or have learned it via +advance.

**Results**
- **10+** Strong hit — full effect.
- **7-9** Weak hit — you succeed, but with a complication (GM's choice).
- **6-** Miss — mark XP. Doom advances if a Dungeon Run is active.

**Examples**
```
+move Hack and Slash
+move Defy Danger
+move Investigate
```

**See also:** +sheet, +roll, +advance
